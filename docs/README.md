# GHEVERHAN Commerce Platform

> Build Simple. Scale Smart.

Modern Headless Commerce Platform dengan Website Builder terintegrasi yang dirancang untuk membangun toko online modern, cepat, dan mudah dikembangkan.

---

## Overview

GHEVERHAN adalah platform e-commerce yang menggabungkan:

- Headless Commerce
- Website Builder
- Content Management System (CMS)
- Admin Panel
- Customer Storefront

Platform ini dirancang untuk memberikan pengalaman belanja premium sekaligus memungkinkan pemilik toko mengelola tampilan website tanpa menulis kode.

---

## Vision

Membangun platform commerce yang:

- Cepat
- Modern

- Mudah dikembangkan
