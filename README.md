# GHEVERHAN Commerce Platform

> Modern Headless Commerce Platform with Integrated Website Builder

---

## Overview

GHEVERHAN adalah platform e-commerce modern yang menggabungkan:

- Headless Commerce
- CMS
- Website Builder
- Admin Panel
- Customer Storefront

Platform ini dirancang untuk memberikan pengalaman belanja yang cepat sekaligus memudahkan pemilik toko mengelola tampilan website tanpa menulis kode.
